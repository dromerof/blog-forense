ADQUISICIÓN DE IMÁGENES FORENSES

La adquisición de imágenes forenses es un proceso para adquirir y guardar imágenes digitales para el análisis y uso en la investigación forense. Específicamente, para restaurar la evidencia digital relacionada con las encuestas penales o privadas, se refiere a obtener imágenes digitales de dispositivos electrónicos como computadora, teléfonos móviles, discos duros, cámaras digitales y otros dispositivos de almacenamiento de datos.

Busca garantizar la consistencia de los datos y almacenar evidencia digital de una manera legalmente aceptable. Se utilizan herramientas técnicas especiales para copiar dispositivos digitales sin alterarlos, por lo que puede realizar un análisis forense sin destruir o cambiar los datos originales.

El recopilado de los datos deben ser precisos y confiables como evidencia de procedimiento legal, es importante obtener una imagen forense correctamente de acuerdo con las directrices y estándares establecidos. El proceso de adquisición utiliza una herramienta confiable para guardar las imágenes resultantes de manera segura y evitar el deterioro.

VENTAJAS Y DESVENTAJAS DE USAR WINDOWS, LINUX Y MAC PARA LA CREACIÓN DE IMÁGENES FORENSES

Las ventajas y desventajas de usar Windows, Linux y Mac para la creación de imágenes forenses pueden variar según las necesidades específicas del usuario. A continuación se presentan algunas de las ventajas y desventajas generales de cada sistema operativo:

Ventajas de Windows

1. Ampliamente utilizado y fácil de encontrar.

2. Gran cantidad de software disponible para la creación de imágenes forenses.

3. Incluye una herramienta forense integrada de creación de imágenes de disco.

Desventajas de Windows

1. Puede ser más vulnerable a ataques de malware y virus.

2. La integridad de las imágenes forenses generadas en este sistema operativo puede estar en duda.

3. Costo asociado con licencias de software y herramientas adicionales.

Ventajas de Linux

1. Conocido por su estabilidad y seguridad.

3. Provee un mayor nivel de control y configuración que otros sistemas operativos.

1. Puede ser difícil de aprender y utilizar para usuarios no familiarizados.

2. Posible incompatibilidad con hardware o controladores específicos.

3. No es tan ampliamente utilizado como otros sistemas operativos, por lo que puede ser más difícil de encontrar herramientas específicas.

ALGORITMO HASH

El algoritmo hash es muy útil en computación forense, ya que se utiliza para calcular valores únicos (hashes) a partir de datos digitales, como archivos o carpetas. Estos valores sirven para verificar la integridad de los datos, y se pueden comparar con los hashes obtenidos de una fuente de confianza para determinar si los datos han sido alterados o manipulados de alguna manera.

También se utilizan algoritmos de hash criptográficos más complejos para reducir el riesgo de tener colisiones al momento de hacer el cálculo . En resumen, el algoritmo hash es una herramienta muy valiosa en la investigación forense, ya que ayuda a garantizar la autenticidad y la integridad de la evidencia digital.

El algoritmo hash es utilizado en computación forense por varias razones, entre ellas:

1. Verificar la integridad de los datos: los valores hash se utilizan para verificar que los datos no han sido alterados o manipulados de alguna manera, lo que ayuda a garantizar la autenticidad y la integridad de la evidencia digital.

2. Identificar archivos duplicados: los hashes se utilizan para identificar archivos idénticos o duplicados en una investigación forense, lo que puede ser útil para encontrar pistas o evidencia adicional.

3. Comparar con una fuente de confianza: los valores hash se pueden comparar con los hashes obtenidos de una fuente de confianza para determinar si los datos han sido manipulados o no.

4. Acelerar la búsqueda de información: los hashes también se pueden utilizar para acelerar la búsqueda de información en bases de datos o en grandes conjuntos de datos, ya que permiten una comparación más rápida y eficiente de los datos.
